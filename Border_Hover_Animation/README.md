# Neon Border Hover Animation

A minimal front-end project showcasing a glowing neon border animation using pure CSS. When the user hovers over the card, the border animates in a circular motion, the shadow intensifies, and the text glows - creating a clean cyberpunk-style effect without any JavaScript.

## How It Works
- A custom CSS property `--rotate` controls the gradient rotation  
- @property enables smooth animation of CSS variables  
- @keyframes animates the border by rotating the conic-gradient  
- The hover state activates the animation + glow effects  
- The card stays clean and static until user interaction
